Portafolio Personal

Esta prueba consiste en el desarrollo de un portafolio personal que reúne los proyectos y trabajos realizados. El portafolio está diseñado para mostrar habilidades en desarrollo web, incluyendo el uso de HTML, CSS y JavaScript. Además, se enfoca en la organización, diseño responsivo y funcionalidad para ofrecer una experiencia de usuario óptima.
